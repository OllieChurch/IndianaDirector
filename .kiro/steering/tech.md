# Tech Stack

## Framework
- Vue 2.7.x (Options API)
- Vue Router 3.x
- Vue CLI 5.x

## UI Framework
- Bootstrap 5.x
- BootstrapVue 2.x (Vue 2 compatible Bootstrap components)

## Build & Tooling
- Babel with Vue CLI preset
- ESLint with `plugin:vue/essential` and `eslint:recommended`
- Webpack (via Vue CLI)

## CMS & Hosting
- NetlifyCMS for content management
- Netlify Identity for authentication
- Netlify Forms for contact submissions
- Netlify for hosting with continuous deployment

## Common Commands

```bash
# Development server
npm run serve

# Production build
npm run build

# Lint and fix files
npm run lint
```

## Path Aliases
- `@/` maps to `src/` (configured in jsconfig.json)

## Browser Support
- Last 2 versions of major browsers
- > 1% market share
- No dead browsers
